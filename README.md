## 🚀 Featured Project

# Firula

**Firula** is an all-in-one event management platform built to simplify the entire event lifecycle — from event creation to ticket sales, attendee management and check-in.

The platform includes:

- 🎟️ Online ticket sales
- 💳 Secure payment processing
- 📱 Mobile check-in with QR Code
- 👥 Public organizer profiles
- ❤️ Social features (followers & attendees)
- 📊 Organization management
- 📢 Push notifications
- 🏟️ Court booking & Day Use reservations
- 📈 Analytics and reporting

Built with:

- React
- React Native (Expo)
- NestJS
- Prisma
- PostgreSQL
- Redis
- Docker
- AWS S3 / Cloudflare R2
